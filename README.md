# 医院管理小程序 python1099

## 项目概述

医院管理小程序是一款旨在提高医疗服务效率，为患者和医生提供便捷服务的平台。通过整合用户管理、预约挂号、就诊记录、药品与器械信息管理等功能，实现医疗信息的数字化和流程的优化。

## 技术栈

- 后端：Python 3.7.7，Django 2.0，pymysql，click，django-threadlocals，requests，xlrd 1.2.0，django-cors-headers
- 前端：Vue 2，uni-app
- 运行环境：MySQL 5.7，Node 12，cnpm 7.1.0

## 功能模块

### 用户管理

- 用户注册、登录
- 个人信息维护
  - 管理病历资料

### 医生管理

- 医生信息注册、更新
- 排班管理

### 预约挂号管理

- 线上预约挂号
- 自动分配就诊号
- 就诊提醒服务

### 科室名称管理

- 科室信息添加、修改、删除

### 就诊记录管理

- 记录就诊历史
- 提供历史信息查询

### 检验信息管理

- 管理检验报告和结果

### 体检档案管理

- 管理体检报告和档案

### 药品信息管理

- 管理药品名称、规格、库存

### 器械信息管理

- 管理医疗器械信息

### 健康提醒管理

- 设置健康提醒

### 论坛交流

- 线上交流平台

### 订单管理

- 管理药品购买、体检预约等订单

## 系统角色

- 患者
- 医生
- 医院管理员
